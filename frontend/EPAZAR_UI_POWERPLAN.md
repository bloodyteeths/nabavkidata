## Database Queries for APIs

### Get evaluation data for tender:
```sql
SELECT
  e.line_number,
  e.item_subject,
  e.offered_brand,
  e.unit_price_without_vat,
  e.winner_name,
  -- Market comparison
  (SELECT MIN(unit_price_without_vat) FROM epazar_item_evaluations WHERE item_subject ILIKE '%' || e.item_subject || '%') as market_min,
  (SELECT AVG(unit_price_without_vat) FROM epazar_item_evaluations WHERE item_subject ILIKE '%' || e.item_subject || '%') as market_avg,
  (SELECT MAX(unit_price_without_vat) FROM epazar_item_evaluations WHERE item_subject ILIKE '%' || e.item_subject || '%') as market_max
FROM epazar_item_evaluations e
WHERE e.tender_id = 'EPAZAR-XXX';
```

### Get bidders for tender:
```sql
SELECT bidder_name, is_winner, is_rejected
FROM epazar_bidders
WHERE tender_id = 'EPAZAR-XXX'
ORDER BY is_winner DESC;
```

### Price intelligence search:
```sql
SELECT
  MIN(unit_price_without_vat) as min_price,
  AVG(unit_price_without_vat) as avg_price,
  MAX(unit_price_without_vat) as max_price,
  COUNT(DISTINCT tender_id) as tender_count,
  offered_brand,
  COUNT(*) as brand_count
FROM epazar_item_evaluations
WHERE item_subject ILIKE '%Ñ‚Ğ¾Ğ½ĞµÑ€ HP%'
GROUP BY offered_brand
ORDER BY brand_count DESC;
```

### Supplier win stats:
```sql
SELECT
  winner_name,
  COUNT(*) as wins,
  SUM(unit_price_without_vat * quantity) as total_value,
  array_agg(DISTINCT item_subject) as products
FROM epazar_item_evaluations
WHERE winner_name ILIKE '%ĞœĞĞ“Ğ ĞĞĞ˜%'
GROUP BY winner_name;
```
