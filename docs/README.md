# nabavkidata Documentation

This directory contains all project documentation organized by category.

## Directory Structure

- **`qa-testing/`** - QA test reports, results, and walkthroughs
- **`security/`** - Security testing, checklists, and vulnerability reports
- **`api/`** - API documentation, guides, and test results
- **`integration/`** - Integration test results and reports
- **`deployment/`** - Deployment guides, AWS setup, and infrastructure docs
- **`branding/`** - Brand assets, guidelines, and design documentation
- **`agent-reports/`** - AI agent summaries and automated reports
- **`guides/`** - User guides, quick starts, and how-tos
- **`project-status/`** - Project summaries, roadmaps, and status reports
