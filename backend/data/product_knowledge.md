# nabavkidata.com - Product Knowledge

## What is nabavkidata.com?
Macedonia's procurement intelligence platform. 270,000+ tenders from e-nabavki.gov.mk and e-pazar.gov.mk. AI-powered search, risk analysis, price intelligence, and competitor tracking for suppliers bidding on government contracts.

## Main Pages & Features

### 1. Табла (Dashboard) - /dashboard
- Personalized feed: recommended tenders matching user's preferences
- AI insights: competitor activity, market trends
- Stats: recommendations count, competitor activities, open tenders
- Onboarding checklist for new users
- Quick access to all sections

### 2. Тендери (Tender Search) - /tenders
- Search 270K+ tenders by keywords (supports Latin and Cyrillic)
- Filters: status (open/closed/awarded/cancelled), category, CPV code, procuring entity, budget range, date range, procedure type
- Sort by relevance, date, value
- Tender cards show: title, entity, value (MKD), deadline, status, match score
- Click tender for full details: documents, bidders, lots, amendments
- Export results as CSV
- Save searches for quick access later
- Default shows only open tenders

### 3. e-Пазар (e-Pazar Marketplace) - /epazar
- Small-value procurement data from e-pazar.gov.mk
- Stats: total tenders, active, awarded, total value
- **Price Check Tool** (AI-powered):
  - Search any product to see historical market prices
  - Shows: min/max/average price, winning brands, sample size
  - AI bidding recommendation with suggested price range
  - Price history visualization chart

### 4. Производи (Products) - /products
- Browse products purchased through tenders
- Search by product name or CPV code
- See prices paid across different tenders

### 5. Добавувачи (Suppliers) - /suppliers
- Browse 40,000+ supplier profiles
- Search by company name (Latin or Cyrillic)
- Stats per supplier: bid count, win count, win rate %, total value won, city, tax ID
- Sort by wins, bids, win rate, name
- Click for detailed supplier profile
- Supplier strength analysis available

### 6. Анализа на Ризик (Risk Analysis) - /risk-analysis
- AI-powered corruption detection across all tenders
- **Flagged Tenders tab**: Pre-analyzed tenders with risk scores (Critical/High/Medium/Low)
- 8 risk indicators: single bidder, repeat winner, price anomaly, bid clustering, short deadline, high amendments, spec rigging, related companies
- Search by institution or winner name
- Filter by risk level and flag type
- Circular risk score visualization with detailed evidence
- **Search tab**: Analyze any tender on-demand with one click
- Download flagged tenders as CSV
- Legal disclaimer: AI analysis for informational purposes

### 7. Конкуренти (Competitor Tracking) - /competitors
- Track competitor companies and their bidding activity
- Search and add competitors to your watchlist
- Per competitor: total bids, wins, win rate, avg bid value, total won value
- Recent wins with tender details
- Common categories (bid/win counts by sector)
- Frequent procurement institutions they bid on
- Head-to-head comparison between competitors
- Product specs and pricing patterns

### 8. Бизнис Анализа (Business Analytics) - /trends
- **Upcoming Opportunities**: Tenders closing soon
- **Price Benchmarks**: Average prices by category
- **Active Buyers**: Top procuring institutions
- **Seasonal Patterns**: When tenders are published by month/season
- Filter by CPV code for industry-specific data
- Market overview: total tenders, market value, open tenders, closing within 7 days

### 9. Алерти (Smart Alerts) - /alerts (or /settings/saved-searches)
- **How to create an alert**: Go to Alerts page > Create Alert tab > Set keywords, CPV codes, budget range, entities > Save
- Three tabs: My Alerts, Matches, Create Alert
- Alert matching: system automatically matches new tenders against your criteria
- Matches show: tender title, match score %, match reasons, deadline
- Email notifications for new matches (instant or daily digest)
- Edit or delete existing alerts

### 10. Inbox (Пораки) - /inbox
- View all alert matches in one place
- Notifications center
- Bell icon in top navigation shows unread count

### 11. AI Асистент (AI Chat) - /chat
- Ask questions in natural language (Macedonian or English)
- **Regular mode**: Ask about any tender, supplier, price, or market data
- **Alerts mode** (bell icon): Analyze your alert matches, check deadlines, compare values
- AI remembers conversation context (persistent memory)
- Sources cited from actual tender documents
- Confidence score shown per answer
- Thumbs up/down feedback
- Suggested questions to get started
- Also available as floating widget on every page (bottom-right corner)
- Widget can be maximized to full chat page

### 12. Поставки (Settings) - /settings
- **Profile**: Edit name, company info
- **Preferences**: Select sectors of interest (IT, Construction, Medical, etc.), CPV codes, budget range
- **Subscription**: View current plan, upgrade, manage billing
- **Saved Searches**: Manage saved search configurations

## Key Workflows

### Search for tenders
1. Go to Тендери page
2. Type keywords in search bar (works in Latin or Cyrillic)
3. Use filters to narrow: status, CPV code, budget range, entity
4. Click a tender for full details and documents

### Create an alert
1. Go to Алерти page
2. Click "Креирај алерт" tab
3. Enter alert name, keywords, CPV codes, budget range
4. Save - system will match new tenders automatically
5. Check Matches tab or Inbox for results

### Check market prices
1. Go to e-Пазар page
2. Use Price Check tool
3. Enter product name
4. See min/max/avg prices and AI recommendation

### Analyze corruption risk
1. Go to Анализа на Ризик page
2. Browse pre-flagged tenders OR search any tender
3. Click "Анализирај" for AI risk assessment
4. Review risk score and specific flags

### Track a competitor
1. Go to Конкуренти page
2. Search for company name
3. View their bids, wins, categories
4. Add to watchlist for monitoring

## Tips
- Search works in both Latin and Cyrillic (type "Alkaloid" or "Алкалоид")
- Use the floating AI chat widget (bottom-right) on any page for quick questions
- Export tender results as CSV from the Тендери page
- The AI assistant can help with any feature - just ask "how do I..."
- Bookmark /dashboard for personalized tender recommendations
- Set your preferences in Settings to improve AI recommendations
